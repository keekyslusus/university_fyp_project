# VPN Config Analyzer

Настольное приложение для анализа безопасности конфигурационных файлов OpenVPN и WireGuard.

## Возможности

- загрузка `.ovpn`, `.conf` или `.txt` файла;
- автоматическое определение OpenVPN/WireGuard;
- поиск потенциально небезопасных параметров;
- расчёт общей оценки безопасности;
- рекомендации по исправлению найденных проблем;
- экспорт отчёта в `.txt`.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Примеры

В папке `samples` есть демонстрационные конфигурации с намеренно небезопасными параметрами.
