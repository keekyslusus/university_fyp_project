# Scandium - VPN config analyzer

Настольное приложение Scandium для анализа безопасности конфигурационных файлов OpenVPN и WireGuard.

## Возможности

- загрузка `.ovpn`, `.conf` или `.txt` файла;
- автоматическое определение OpenVPN/WireGuard;
- поиск потенциально небезопасных параметров;
- расчёт общей оценки безопасности;
- рекомендации по исправлению найденных проблем;
- экспорт отчёта в `.txt`.

## Запуск

```bash
npm install
npm run dev
```

## Анализ от ИИ

Для блока «Анализ от ИИ» нужен Gemini API key. Scandium использует официальный SDK `@google/genai` и модель `gemini-2.5-flash`. Ключ можно ввести прямо в приложении или создать `.env` файл:

```bash
VITE_GEMINI_API_KEY=ваш_ключ
```

Результат Gemini кэшируется по SHA-256 хэшу содержимого файла, поэтому один и тот же конфиг повторно не отправляется. Ответ хранится в структурированном виде: краткий вывод, ключевые риски, рекомендуемые действия и итог.

Документация SDK: https://googleapis.github.io/js-genai/release_docs/index.html

```bash

```

## Примеры

В папке `samples` есть демонстрационные конфигурации с намеренно небезопасными параметрами.

## Структура

- `src/core` — парсинг, правила анализа, отчёты, Gemini-кэш и AI-запросы.
- `src/components` — UI-компоненты приложения.
- `src/i18n` — словари интерфейса и провайдер локализации.
- `src/utils` — небольшие общие утилиты.
- `electron` — запуск настольного окна Electron.

## Локализация

Переключение языка доступно в Electron-меню `View -> Language`. Язык по умолчанию — английский. Основные строки интерфейса вынесены в `src/i18n/dictionaries.ts`. Тексты правил анализа и экспортируемого отчёта пока остаются отдельным слоем локализации.
